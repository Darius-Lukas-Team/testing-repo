# Testing Repo - Claude Cowork Plugin Marketplace

A test marketplace for validating the Claude Cowork plugin installation flow.

## Installation

1. Open **Claude app > Cowork > Customize > Plugins > + > Add marketplace**
2. Enter `Darius-Lukas-Team/testing-repo`
3. Click **Sync**
4. Browse and install any of the plugins below

## Plugins

### Sales Copywriter
Generate ad hooks, audit sales pages, and position offers.

**Skills:**
- `ad-hooks` — Generate 3 punchy ad hooks
- `sales-page-audit` — Give 3 quick wins to improve a sales page
- `offer-positioning` — Suggest a positioning statement

### Content Engine
Create YouTube shorts scripts, LinkedIn posts, and repurpose blog content.

**Skills:**
- `youtube-shorts` — Write a short-form video script idea
- `linkedin-authority-post` — Write a LinkedIn post outline
- `blog-repurposer` — Suggest 3 ways to repurpose a blog post

### Client Acquisition
Diagnose funnels, build low-ticket offers, and design upsell stacks.

**Skills:**
- `funnel-diagnosis` — Identify the weakest funnel stage
- `low-ticket-offer-builder` — Suggest a low-ticket offer concept
- `upsell-stack-designer` — Suggest an upsell sequence

### Design Slides
Generate scrollable HTML slide pages optimized for screen-recording video content (YouTube, VSL, Webinar).

**Skills:**
- `design-slides` — Create polished single-file HTML slide pages from an outline. Supports YouTube videos, Video Sales Letters, and Webinar presentations. Output saves to your Desktop and auto-opens in your browser.

**Requires Claude Code enabled in your Claude app.**
