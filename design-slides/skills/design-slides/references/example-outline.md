## YouTube Video Example

```
VISUAL OUTLINE: n8n Master Workflow Architecture
Video type: YouTube
Estimated sections: 8 (for ~12 min video)

1. Hero
   Component: Hero Section
   Content summary: Title "n8n Master Workflow Architecture" with subtitle about
   automating promo campaigns

2. About Me
   Component: About Me (fixed — copy from templates/sections/about-me.html)

3. The Architecture Decision
   Component: Comparison Grid (3 columns)
   Content summary: Compare three approaches — one giant workflow vs. three
   separate vs. master + sub-workflows. Recommended badge on the third.

4. Master Workflow — Phase 1
   Component: Section Container + Node Row
   Content summary: Manual Trigger → Fetch Promo DNA → Fetch Frameworks → Split
   to sub-workflows. Show as connected nodes.

5. Sub-Workflows Overview
   Component: Card Grid (3 columns)
   Content summary: Invite, Show-Up, and Sales sequence cards with branch lists,
   asset counts, and Claude call counts.

6. How the Pieces Connect
   Component: Flow Diagram + Callout Box
   Content summary: Visual showing master passes context to subs, subs return
   outputs. Callout explaining the "Execute Sub-Workflow" n8n node.

7. Output Phase
   Component: Node Row
   Content summary: Merge → Notion Campaign Hub → Slack Notification

8. By the Numbers
   Component: Stat Cards
   Content summary: 4 workflows, 16 Claude calls, 113 assets, ~2 min runtime

9. Key Takeaways
   Component: Summary/Recap Slide
   Content summary: Run full campaign in one click, run any sequence solo,
   add new sequences without touching existing ones, error isolation.

10. Your Next Step
    Component: YouTube CTA (fixed — copy from templates/sections/next-step-youtube.html)
```

## Phase 2 Notes

When building from an approved outline:

- Each numbered section becomes one visual section on the page
- The component type maps directly to the patterns in `components/components.md`
- Section titles become the `section-title` text
- Content summaries become the actual data inside each component
- Sections should flow naturally when scrolled top-to-bottom
- Add section dividers between major topic shifts, not between every section
- Fixed sections (About Me, Next Step, scroll nav) must be copied exactly from their template files
